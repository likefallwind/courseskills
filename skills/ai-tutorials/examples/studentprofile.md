# 学生画像介绍

这是一个用于 AI 教程和课程设计的标准学生画像示例。在设计课程或生成教学内容时，提供清晰的学生画像可以帮助 AI 更好地调整内容的深度、术语和教学节奏。

## 1. 学生身份 (Student Profile)
* **教育背景**：非计算机专业本科生，可能来自历史、金融、生物、物理等方向。
* **学习目标**：希望利用 Vibe Coding 把自己的想法做成可运行的小工具或作品，能借助 AI 解决专业学习和日常工作中的实际问题。
* **学习风格**：实践导向（喜欢边做项目边理解概念），对抽象术语和大段理论容易失去耐心，更喜欢通过具体案例、图解和可运行结果建立信心。

## 2. 已掌握知识与课程 (Prerequisites & Current Skills)
* **编程语言**：几乎零基础，可能只接触过少量 Python 基础语法，不能独立完成完整程序。
* **数学基础**：差异较大，不应默认掌握线性代数、概率统计或复杂公式推导。
* **计算机基础**：不了解系统的数据结构、算法、网络、数据库等计算机专业知识。
* **开发工具**：可能使用过普通文本编辑器或在线工具，但不熟悉 Git、命令行、IDE、包管理器和本地开发环境。

## 3. 兴趣点与探索方向 (Points of Interest)
* **个人作品开发**：希望把个人介绍页、学习工具、资料整理工具、可视化小应用等想法快速做出来。
* **效率与自动化工具**：希望用 AI 和脚本处理重复劳动，例如整理文件、清洗表格、生成报告、汇总资料。
* **专业场景落地**：希望把 Vibe Coding 用到自己的专业方向，例如历史资料可视化、金融数据分析、生物实验记录整理等。
